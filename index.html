About Me

My full name is Joe Grace III. I have always gone by the nickname Trey, which means three. I am originally from Houston, TX. After high school I joined the Ohio Army National Guard. In 2012, I deployed to Bagram Air Field in Afghanistan. After returning from deployment, I started working as a CNC machinist. I worked in this field from 2013-2020. I always liked getting to solve problems and seeing something I made come to life. When I decided I wanted to get into tech, I thought my machinists skills would transfer to a Developer role and I could solve different problems in a different way. In summer of 2020, I attended We Can Code IT's full stack Java bootcamp to help achieve my goal of becoming a Software Developer. When I am not trying to further my skills, I enjoy spending time with my wife and our two dogs, Sam and Zoey. Some of my interests include hunting, video games, DIY home improvement projects, and smoking various meats.

My Projects

VirtualPet Project:

The goal for this project was to make a Virtual Pet(ie. tamagotchi). My animal was a tiger cub inspired by the Tiger King TV show. This was my first Java project. I used various int variables to keep track of health, thirst, hunger, boredom, and tiredness. For the console output, I used a while loop with minimum levels the user has keep the animals stats above to keep the animal "alive". A "tick" was created that runs after every turn to make the pet's needs increase. I used a random number generator for a random chance the animal gets sick.
GitHub Repository

VirtualPetShelter Project:

This project built on the VirtualPet project by having multiple pets. This was my first project done using TDD. The pet class was almost the same with a Shelter class which housed all the pet objects using a Map. The Shelter class also called methods for every pet and even made a method for calling a single individual pet given it's name.
GitHub Repository

VirtualPetAmok Project:

This project built on the Shelter project by having multi different kinds of animals all in one shelter. This project was completed using TDD. This project utilizes encapsulation, inheritance, polymorphism, and TDD. I also used instanceof cases to select between the different kind of animals in the shelter.
GitHub Repository
